Minecraft Plugin

Fastbuild

FastBuild is an plugin that help you accelarate build/break move in Minecraft.
  • Break at most 64 same blocks at once.
  • Build at most 64 same blocks with one clicks.

    For now, Fastbuild supports multiplayer server running on Bukkit or Spigot.

    It make no difference with Minecraft default mode when not in use. The administrator has the right to control who can use FastBuild functions. Play speaks louder than word. Just install and try it.

    Install Steps:

    Three-Step Install:

    Step 1. Download FastBuild and put fastbuild.jar file into your "plugins/" folder

    Step 2. Either give op or, if you have a permissions plugin, give the permission nodes.

    Step 3. Run your server!

    !!!NOTICE!!!

    WATCH OUT SUFFOCATING OR FALLING FROM HIGH PLACE WHEN BUILDING WITH N NOT EQUAL TO 1

    Permissions:

    There are two permissions to use fastbuild:

    * FastBuild.setn: true

    * FastBuild.basic: true

    Now you can use 'setn', and build/break n blocks with one build/break.
